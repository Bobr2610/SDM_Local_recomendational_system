Local recommendations system
Кейс: Персональные предложения банковских продуктов

Мы хотим сделать полностью автономную Edge модель для использовании прямо на телефоне юзера, тем самым убирая необходимость отправлять данные пользователя на сервер и выделять дополнительные мощности под систему рекомендаций.

Данные будем брать из трех публичных датасетов клиентов
1. Bank Customer Churn Dataset
2. Credit Card Customers Dataset
3. UCI Bank Marketing Dataset
И гененировать дополнительные синтетические фичи, рандомные с заданными вероятностями.
После этого планируется сделать синтетические аггрегированные признаки о транзакциях клиентов за последние 3-6 месяцев.

Модель будет выдавать 5 самых релевантных пользователю продуктов для дальнейшей рекомендации их в мобильном приложении.

Оценивать будем модель по следующим метрикам
1. Precision@
2. NDCG@
3. CTR simulation
4. Business value*

*Условная бизнес-ценность рекомендаций для банка.
Каждому офферу задаем profit_score от 1 до 10.

Таким образом, планируем получить модель которая выдает качественные персонализированные рекомендации полностью локально на телефоне пользователя
